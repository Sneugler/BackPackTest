# ToolkitPlanner Module

ToolkitPlanner produces an authorized-use tooling plan for a project by mapping platform capabilities to assessment phases and filtering recommendations through the project's rules of engagement.

Outputs include:
- ready vs blocked recommendations
- reasons when a recommendation is blocked by ROE
- command/module references so operators can move directly into approved workflows
- validation-path plans with entry methods, prerequisites, evidence expectations, and guardrails for controlled exercises
- path-of-least-resistance ranking for quick prioritization
- MITRE ATT&CK technique references and defensive-method mappings for each path
- exportable JSON/Markdown evidence packages for report-ready documentation
