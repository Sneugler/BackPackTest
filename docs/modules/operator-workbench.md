# OperatorWorkbench Module

OperatorWorkbench aggregates project status, recent events, quick actions, and saved views for day-to-day operator use.

Related safe operator surfaces now include:
- a project findings dashboard view via `GET /projects/{project_id}/findings`
- a portfolio rollup via `GET /portfolio/summary`
- CLI access through `aegisforge operator findings <project_id>` and `aegisforge project summary`
