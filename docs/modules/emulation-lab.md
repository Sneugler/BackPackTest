# EmulationLab Module

EmulationLab is a first-class authorized red-team and purple-team planning module. It structures scenarios, telemetry expectations, control validation, and observed-vs-expected review templates.

## Critique pass
- **Architecture**: Encapsulated in a dedicated module package.
- **UX**: Added review templates to reduce operator overhead during after-action work.
- **Output quality**: Gap analysis now derives from expected telemetry for consistent validation plans.
- **Operator usefulness**: Keeps adversary emulation focused on controlled exercises, not offensive automation.
